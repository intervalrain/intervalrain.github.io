[ML] Start Tensorflow Environment with Conda

October 11, 2023 · 2 分鐘 · Rain Hu | Suggest Changes
目錄
  • 環境建置
  • For terminal user

環境建置#

  1. 安裝 Anaconda
  2. 創建虛擬環境
  1. 進入虛擬環境 (macOS/Linux)
  1. 在環境內安裝 tensorflow
  1. 在環境內安裝 jupyter notebook
  1. 在環境內安裝 pandas
  1. 開啟 jupyter notebook

For terminal user#

  1. 開始 Anaconda.Navigator
  2. 在 Environments 中安裝指定模組 ex.tensorflow, keras
  3. 在 terminal 中輸入 conda activate {環境名稱}
  1. 開啟 python

若成功便會顯示 python 安裝資訊
