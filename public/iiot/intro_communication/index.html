通訊理論#

通訊基礎概念#

什麼是通訊?#

通訊(Communication) 本質上是訊息傳遞的過程。在電子系統中，通訊是指兩個或多個裝置之間交換數位資訊的過程。

基本通訊模型#

    B[Encode]

    C --> D
    D --> E
    E --feedback/ack--> A

核心概念#

1. 訊號與雜訊

  • 訊號 (Signal): 攜帶有用資訊的電氣變化
  • 雜訊 (Noise): 干擾訊號傳輸的無用變化
  • 訊雜比 (SNR): 訊號功率與雜訊功率的比值，決定通訊品質。

2. 頻寬 (Bandwidth)

  • 定義: 系統能傳輸的頻率範圍
  • 影響: 決定資料傳輸速率的上限
  • 單位: 赫茲 $Hz$ 或位元每秒 $bps$

3. 延遲 (Latency)

  • 傳播延遲: 訊號在媒介中傳播所需時間
  • 處理傳遲: 裝置處理資料所需時間
  • 佇列延遲: 裝料在緩衝區等待的時間

物理層 (Physical Layer)#

物理層是通訊系統的最底層，負責將數位資料轉換為可在實體媒介上傳輸的訊號。

訊號表示方法#

1. 數位信號

  • 高電位 (High) = 邏輯 1 = 通常 +3.3V 或 +5V
  • 低電位 (Low) = 邏輯 0 = 通常 0V 或 -V

2. 編碼方式

不歸零碼 (NRZ - Non-Return-to-Zero)

  • 優點：簡單，頻寬利用率高
  • 缺點：無時鐘訊息，難以同步

曼徹斯特編碼 (Manchester)

  • 優點：自同步，容錯性好
  • 缺點：需要雙倍頻寬

傳輸媒介#

1. 銅線傳輸

  • 特性阻抗：通常 50Ω 或 75Ω
  • 信號衰減：距離越長衰減越大
  • 串音：相鄰線路間的電磁干擾

雙絞線原理：

2. 光纖傳輸

  • 全反射原理：光在光纖核心與包層界面發生全反射
  • 模態色散：不同路徑的光有不同傳播時間
  • 材質色散：不同波長在材質中的傳播速度不同

3. 無線傳輸

  • 電磁波譜：從無線電波到可見光
  • 調變技術：AM、FM、PSK、QAM
  • 多重存取：TDMA、FDMA、CDMA

信號品質考量#

1. 失真 (Distortion)

2. 衰減 (Attenuation)

  • 公式：Loss (dB) = 20 × log₁₀(Vout/Vin)
  • 補償：放大器、中繼器

3. 反射 (Reflection)

  • 阻抗不匹配造成信號反射
  • 影響：信號品質惡化、功率損失

資料鏈結層 (Data Link Layer)#

資料鏈結層負責在物理連線上提供可靠的資料傳輸。

訊框結構 (Frame Structure)#

各部分功能：

  1. 前導碼 (Preamble)：幫助接收端同步時鐘
  2. 開始碼 (SOF)：標記訊框開始
  3. 標頭 (Header)：包含地址、控制資訊
  4. 資料 (Data)：實際要傳輸的資訊
  5. 檢查碼 (CRC)：錯誤檢測
  6. 結束碼 (EOF)：標記訊框結束

錯誤檢測與修正#

1. 同位檢查 (Parity Check)

  • 檢測能力：單一位元錯誤
  • 限制：無法檢測偶數個錯誤

2. CRC (Cyclic Redundancy Check)

CRC 的數學原理：

  • 將資料視為多項式
  • 除以生成多項式得到餘數
  • 餘數即為 CRC 碼

3. 漢明碼 (Hamming Code)

  • 不僅能檢測錯誤，還能修正單一位元錯誤
  • 冗餘位數：log₂(n) + 1 (n 為資料位數)

流量控制 (Flow Control)#

1. 停止等待 (Stop-and-Wait)

  • 優點：簡單可靠
  • 缺點：效率低，不適合高延遲環境

2. 滑動視窗 (Sliding Window)

  • 優點：高效率，充分利用頻寬
  • 複雜度：需要緩衝管理

媒體存取控制 (MAC)#

1. CSMA/CD (載波感測多重存取/碰撞檢測)

2. Token Ring (權杖環)

  • 權杖在環上循環
  • 只有持有權杖的節點才能傳輸
  • 確定性延遲，無碰撞

結論#

通訊技術從最基本的電氣信號開始，透過層層抽象和封裝，發展成為現今複雜的網路系統。理解這個演進過程有助於：

  1. 系統性思維：從整體角度理解通訊系統
  2. 問題診斷：能夠在正確的層次定位問題
  3. 技術選擇：根據需求選擇合適的通訊技術
  4. 未來準備：掌握技術發展趨勢和基本原理

工業通訊作為通訊技術的重要應用領域，具有其特殊的需求和挑戰。隨著工業 4.0 和物聯網的發展，傳統工業協定與現代資訊技術的融合將是未來的重要趨勢。

掌握從物理層到應用層的完整知識體系，不僅能幫助我們更好地理解現有系統，也能為設計未來的通訊解決方案提供堅實的理論基礎。
